## 공부한 내용 본인의 언어로 정리하기
``` java
public static int[] 자연수_뒤집어_배열로_만들기(long n) {
    String str = String.valueOf(n);

    int[] answer = new int[str.length()];

    for (int i = 0; i < str.length(); i++) {
        answer[i] = Character.getNumericValue(str.charAt(str.length() - i - 1));
    }

    return answer;
}
```
