## 공부한 내용 본인의 언어로 정리하기
``` java
boolean solution(String s) {
   boolean answer = true;
      char[] chars = s.toCharArray();
      int p = 0;
      int y = 0;
      for (char c : chars) {
    if(c == 80 || c == 112) {
      p++;
    }else if(c == 89 || c == 121) {
      y++;
    }
  }
      
      if(p != y) {
        answer = false;
      }
     
      return answer;
}
```
## 오늘의 회고

- 어떻게 해결했는지

- 무엇을 새롭게 알았는지
  - 새롭게알게된건 없고 오랜만에 아스키코드 찾아봤다
  - 다른사람들 풀이 보니까 재밌었고, 스트림이랑 람다로 한 걸 보니 훨씬 좋은 풀이같았다
